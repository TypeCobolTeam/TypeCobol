    88  check       VALUE 'T'.

PROCEDURE DIVISION.
*******************
TEST-Bool.
  MOVE        check     TO   trtOk

* MOVE        true     TO   check
  set check to true
* MOVE        false     TO   check
  set check-false to true

  SET check to TRUE
* Set check to false
  Set check-false to false
  
  if check
    continue
  end-if
*  if check = true
  if check
      continue
  end-if
  if not check
      continue
  end-if
*  if check = false
  if check-false
      continue
  end-if
	evaluate true
		when check
		 continue
		when trtOk
		 continue
*		when check = false
        when check-false
		 continue
    end-evaluate
